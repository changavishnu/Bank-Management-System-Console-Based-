Bank Management System (Java)

 Overview

The Bank Management System is a simple Java-based project for beginners that demonstrates basic banking operations such as deposit and withdrawal. It helps understand the fundamentals of object-oriented programming, modular code design, and user interaction through the console.


---

Features

Create a new account

Deposit money

Withdraw money

Check balance

Simple method-based structure (no collections or arrays used)

How to Run

1. Clone the repository:

git clone https://github.com/changavishnu/bank-management-system.git


2. Open the project in any IDE (like IntelliJ IDEA or VS Code or Command Prompt).


3. Compile - > javac -d . Main.java

   Run - > java com.bankmanagement.java
